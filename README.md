# Hello World App

## Описание проекта

Простое веб-приложение, отображающее текст «Hello World» и две кнопки для изменения цвета текста на зелёный и оранжевый.

## Используемые технологии

### Frontend

В проекте используются следующие технологии:

* HTML — создание структуры веб-страницы;
* CSS — оформление интерфейса;
* JavaScript — реализация логики изменения цвета текста.

### Framework

Фреймворки в данном проекте не используются, так как для решения поставленной задачи достаточно стандартных возможностей HTML, CSS и JavaScript.

### Backend

Backend отсутствует.

Приложение является статическим и работает полностью в браузере пользователя. Для выполнения задания не требуется серверная часть, база данных или хранение данных.

## Причины выбора технологий

* HTML обеспечивает структуру страницы;
* CSS отвечает за внешний вид элементов;
* JavaScript реализует обработку нажатий на кнопки;
* отсутствие Backend упрощает разработку и развертывание приложения.


## Возможные уязвимости приложения

При публикации приложения в открытом доступе могут возникнуть следующие риски:

### 1. Изменение кода пользователем

Поскольку приложение работает в браузере, пользователь может открыть инструменты разработчика и изменить код страницы. Для данного проекта это не критично, так как приложение не хранит важные данные.

### 2. Отсутствие защищённого соединения

Если разместить приложение без HTTPS, передаваемые данные могут быть перехвачены. Поэтому рекомендуется использовать защищённое соединение.

### 3. Возможные ошибки при добавлении новых функций

В текущей версии приложение не принимает данные от пользователя. Однако при дальнейшем развитии проекта необходимо проверять вводимые данные, чтобы избежать проблем с безопасностью.

### 4. Публикация служебной информации

При загрузке проекта в публичный репозиторий важно не размещать пароли, ключи доступа и другую конфиденциальную информацию.

## Вывод

Текущая версия приложения является простой и не содержит серьёзных уязвимостей, так как не использует сервер, базу данных и не обрабатывает пользовательские данные.
